# 特殊字符 5：分组

在范围的那一节中，有一个练习是匹配 cat 和 car 这两个单词，当时给的答案是cat|car cat 和 car 这两个单词的前面都是 ca，所以有没有可能提取前面公共的部分 ca，然后用t|r来匹配最后的字母呢？

# 分组

如果你直接写cat|r，发现它是错误的。那我们应该怎么写呢？只需要将t|r用括括起来就可以了。 括号在正则中可以用于分组

符号 含义
(x) 匹配 'x' 并且记住匹配项，其中括号被称为捕获括号
试一试：匹配单词
Task Text Result
match cat
match car
ca(t|r)
查看答案。

被括号括起来的部分“子表达式”会被保存成一个子组。

试一试：匹配单词，捕获子组
Task Text Capture Groups Result
capture cat t
capture car r
ca(t|r)
查看答案。
练习 1：提取图片文件名和后缀
Task Text Capture Groups Result
capture test.png testpng
capture test123.jpeg test123jpeg
capture aaa.jpeg aaajpeg
skip aaa.pdf
(\w+).(png|jpeg)
查看答案。

# 引用子组

我们怎么引用被保存的子组呢？我们可以用\n我们引用第 n 个括号的内容。

符号 含义
\n 返回第 n 个子捕获匹配的子字符串
试一试：引用子组
Task Text Capture Groups Result
capture cat last letter is t t
capture car last letter is r r
ca(t|r) last letter is \1
查看答案。
练习 2：查找重复的单词
Task Text Capture Groups Result
capture I have a cat cat. cat
capture I have a dog dog. dog
skip I have a dog
(\w+) \1
查看答案。

# 不保存子组

在括号里面的会保存成子组，但有些情况下，你可能只想用括号将某些部分看成一个整体，后续不用再用它，类似这种情况，在实际使用时，是没必要保存子组的。这时我们可以在括号里面使用 ?: 不保存子组。

符号 含义
(?:x) 匹配 'x' 但是不记住匹配项
试一试：只提取图片的文件名
Task Text Capture Groups Result
capture test.png test
capture test123.jpeg test123
capture aaa.jpeg aaa
skip aaa.pdf
(\w+).(?:png|jpeg)
查看答案。

# 括号嵌套

比如在括号嵌套的情况里，我们要看某个括号里面的内容是第几个分组怎么办？

我们只需要数左括号是第几个，就可以确定是第几个子组。

试一试：匹配嵌套组
Task Text Capture Groups Result
capture Jan 1987 Jan 19871987
capture May 1969 May 19691969
capture Aug 2011 Aug 20112011
((:?[A-Za-z]+) (\d+))
查看答案。
